JIGUANG SDK for Android 简介
===========================
JIGUANG SDK for Android 是一套由极光提供的软件库，使开发者可将极光的开发者服务集成到自己的 Android app 中。

关于 JIGUANG SDK for Android 各种的集成，请查阅我们详细的集成指南和 API 指南。
也可参考下载的压缩包中附带的示例 jiguang-demo 来通过代码学习本 JIGUANG SDK 的用法。


新手指南
--------
0. 参考 “集成指南.md” demo 集成部分，快速把 Demo 跑起来，体验 JIGUANG SDK 的服务。
1. 组包服务下载的 JIGUANG SDK，均已集成到 jiguang module 中，参考 “集成指南.md” 快速接入服务。
3. 根据你的业务来构建各服务策略。基础 api 教程见 demo 实例，更多 api 文档见下面文档导航。
